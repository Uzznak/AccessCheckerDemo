
# AccessChecker

AccessChecker est une application web permettant de vérifier rapidement les accès attribués à un utilisateur.

Ce projet a été réalisé comme démonstrateur pour explorer le développement web, la gestion des accès et le déploiement d'applications.

## Fonctionnalités

- Recherche d'un utilisateur
- Affichage des accès associés
- Interface responsive avec Bootstrap
- Déploiement sur Vercel

## Technologies utilisées

- HTML5
- CSS3
- Bootstrap 5
- JavaScript

## Aperçu

L'utilisateur saisit un nom d'utilisateur puis clique pour Vérifier l'accès de la personne.
